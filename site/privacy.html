Privacy without fine print.

The storefront is static and intentionally low-data. It helps you choose and download a plugin without learning who you are.

No analytics · no cookies · no accounts · no browser storage · no third-party scripts or remote fonts.

What the page does

Your host choice lives only in the open page and disappears when it closes. Public catalog refreshes use same-origin data.json. No cross-origin request happens until you follow an external link.

Downloads and plugins

ZIPs are static, same-origin downloads. The Foundry does not learn which host or plugin you choose. Shipped hooks make no network calls and receive no Foundry credential.

What GitHub receives

GitHub Pages necessarily receives normal web request data under GitHub's Privacy Statement. This project does not receive or retain Pages access logs.

Public contributions

Issues and pull requests are public. Never include secrets, private code, customer data, or unnecessary personal information. Report sensitive security matters through private vulnerability reporting.
