# briefing agent — prompt
Assemble the per-shift briefing (tools/briefing.py produces the substance from
repo data). Keep it to a <30-second read: the last line move, the quality number,
the top 3 ranked desk items, any open alarm. Post to the pinned briefing issue.
Every figure must be substantiated by the repo (growth-honesty). Do not invent
movement; a quiet shift says so.
